# WPFStudy

#### 介绍

此库内的所有项目均作为学习使用，不对外开放

此项目用于学习WPF框架与WPFUI组件，使用Visual Studio 2026，SDK使用.NET10

#### Issue

###### 标签说明

1. **`bug`**
   
   用于标记代码中存在的**错误、缺陷或非预期行为**。这类问题通常需要优先修复。

2. **`feature` / `feature request`**
   
   代表一个新的**功能请求或增强提案**。这类Issue常用于收集新想法，并可能转化为开发任务。

3. **`documentation`**
   
   表示该问题与**文档的编写、更新或修正**相关，比如补充缺失的说明或修改过时的内容。

4. **`enhancement`**
   
   指对**现有功能的改进或优化**，目的是提升效率或体验，但并不引入全新的功能。

5. **`good first issue`**
   
   这个标签旨在**帮助新贡献者快速融入项目**，它标记了那些相对简单、适合入门的问题。

6. **`help wanted`**
   
   表明项目维护者**希望在这个问题上获得外部帮助**，是向社区发出协作邀请的信号。

7. **`duplicate`**
   
   标记该问题与已有的另一个问题**重复**。被标记后，通常会被关闭，并将讨论集中到原始问题上。

8. **`invalid`**
   
   这个 Issue 报告的内容**不成立、不符合事实或与项目范围无关**。

9. **`question`**
   
   这个 Issue **不是一个功能请求或 Bug 报告，而是一个需要解答的疑问**

10. **`wontfix`**
    
    维护者**承认这个 Issue 描述的情况可能存在，但决定不为此采取行动**（不修复、不实现）。
